## usersテーブル

|------|----|-------|
|email|string|null: false|
|password|string|null: false|
|username|string|null: false|

- has_many :comments


## groupsテーブル
|Column|Type|Options|
|------|----|-------|
|group|string|null: false|
|member|string|null: false|

- has_many :users
